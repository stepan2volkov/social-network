# Проект "Социальная сеть"
---
### Описание

Приложение для создания и просмотра анкет в социальной сети. 

Позволяет авторизоваться по паролю и просматривать анкеты других пользователей.
Есть возможность подписаться на другого пользователя. При обоюдной подписке пользователи становятся "друзьями".

База данных не выбирается, есть требование использовать MySQL.

Авторизация выполнена в виде jwt-токенов. Способ имеет как свои достоинства, так и недостатки, о чем подробно можно почитать в интернете.

## Инструкция по локальному запуску приложения

Для запуска требуется, чтобы на компьютере был установлен go >= 1.18

1. Поднять контейнер с базой данных

```bash
docker-compose up -d
```

2. Применить к базе данных миграции

```bash
make install-tools & make local-migrate
```

Для миграций используется [goose](https://github.com/pressly/goose).

3. Запустить приложение

```bash
make run
```

При локальном запуске настройки приложения берутся из файла `local.env`. 
Пример настроек умышленно оставлен в репозитории, чтобы можно было быстрее развернуть локально.
Приложение будет доступно на порту `8000`.

```
.
|____cmd
| |____social-network (точка входа в приложение)
|____migrations (миграции для базы данных)
|____httprequests (запросы для тестирования)
|____internal
| |____app (бизнес-логика)
| |____config (получение конфигурации приложения)
| |____api (http-интерфейс приложения)
| | |____server (запуск сервиса)
| | |____openapi (сгенерированный код на основанни спецификации OpenAPI)
| | |____router (роутинг запросов в бизнес-логику)
| |____entities (доменные сущности)
| | |____token
| | |____user
| |____store (реализации хранилищ)
| | |____mysqlstore (реализация хранения MySQL)
|____api (спецификации OpenAPI)
|____build
| |____Dockerfile
```

## Тестирование

Для тестирования можно использовать запросы из папки `httprequests`.
Для работы с файлами запросов из VS Code требуется установить расширение [REST Client](https://marketplace.visualstudio.com/items?itemName=humao.rest-client). Продукты JetBrains поддерживают работу с файлами `*.http` из коробки.
